How Signing Works
  1. Your file is hashed with SHA-256 to produce a unique fingerprint.
  2. The hash is signed with your RSA private key using the PSS padding scheme.
  3. A .sig.json file is generated containing the signature and your public key.
  4. Anyone with the original file + .sig.json can verify authenticity.
